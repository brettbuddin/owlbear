# Owlbear

A temperamental dungeon generator.

## Dependencies

- [Go](https://golang.org)
- [GB](https://getgb.io/)

## Testing and Building

```
$ gb test -v
$ gb build
```

## Usage

```
$ bin/owlbear --help
Usage of bin/owlbear:
  -height int
        height of the dungeon (default 50)
  -max-attempts int
        maximum number of attempts at creating a room (default 20)
  -min-room-dim int
        minimum room diminsion (default 3)
  -min-room-spacing int
        minimum spacing between rooms (default 5)
  -rooms int
        number of rooms (default 20)
  -seed int
        random seed (useful in debugging)
  -width int
        width of the dungeon (default 100)
```
